Accessibility check for Designers

Link to site being evaluated

Checklist item Checked or not Notes
Make sure there is enough contrast between text and its background color. No Poor contrast on page announcement for Michigan Union rennovation. Specifically text that says "We're Moving"
Don't indicate important information using color alone No Hyperlinks for quicklinks section and navigation bar submenu options only change color on hover. Colorblind users or may not notice an element is a hyperlink or not because it does not add an additional affordance like an underline.
Pair values of colors together (not only hues) to increase contrast. Yes
Don't rely on sensory characteristics as the sole indicator for understanding and operating content. No On the transportation info page, links to different transportation options are displayed in a grid of images with a title describing the type. This information is incomplete and titles like "Text Directions" along with the thumbnail pictures are not complete enough to convey meaningful understanding.
Design focus states to help users navigate and understand where they are. No Most focus states are obvious and allow the page to be navigated from the keyboard. However, tabs are not immediately obvious when selecting submenu options for the nav bar.
Write good alt text for your images. Yes
If an experience cannot be made accessible, create another route for users to get that information. Yes
Be as consistent and clear as possible in layout and copy. No No consistency in information layout and type. Quick links are displayed in a list, while links above are displayed with images in a grid. Below that featured events are displayed in a rolling list with images. There is no discernable differences between the first two types of information and why events and feautred events are different. Additionally, the images for the featured events are too distorted from zooming in, making it difficult to see the images.

Recommendations

1. Change hover attributes and style of navigation bar submenu options and quick links to include some sort of additional visual cue besides color change to represent that the element leads to another page. Add an underline or create an animation of the text moving to represent some visual difference between the link from plain text.

2. Include a dropdown menu or extra descriptive text to each option on the transportation links page to offer more insights about what each link leads to

3. Add submenu navbar options to the tabindex to allow for navigation using keyboard.

4. Change layout and design of information links on the main page to be more distinct. Style all options in a grid format with a thumbnail picture to keep consistent and make sure people don't gloss over the quicklinks
